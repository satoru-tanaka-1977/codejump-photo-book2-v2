# CodeJump PHOTO BOOK2-v2

CodeJump入門課題「PHOTO BOOK2」を再現実装しました。  
Flexboxを使用したレイアウト構築と、レスポンシブ対応の練習を目的としています。

---

## 使用技術
- HTML5
- CSS3（Flexbox）
- レスポンシブ対応（max-width: 1024px）
- ress.css（リセットCSS）

---

## 実装内容
- header / main / section / footer を用いた構造
- 画像リストをFlexboxで横並び → SPで縦並び
- DETAILセクションをdl要素で実装
- 1024px以下でレイアウト切り替え
- container / inner による中央寄せ管理
